Conhecimentos

HTML5

A Linguagem de Marcação e Hipertexto foi meu primeiro contato com a tecnologia. Adquiri conhecimento regular no IFBA... Fiz outros cursos envolvendo essa liguagem e a minha principal referencia nesse quesito é o W3.org.

CSS3

O famoso Cascade Style Sheet é o responsável pela maquiagem do HTML. Gosto de experimentar e conhecer novas propriedades e me dedico a isso quando estou em algum projeto, mas sou fã do convencional e do prático quando não temos tanto tempo para ousar.

JavaScript

Tão certo quanto: "Git não é Github", Javascript não é Java. Embora de nomenclaturas parecidas, o JS para os intimos, é uma liguagem de script extremamente poderosa que tem se destacado no cenário front-end mundial ja há alguns anos. Minha principal referencia no assunto é o MDN Web Docs.

SQL

Comumente chamada de Structured Query Language, o SQL é a linguagem dos bancos de dados relacionais e a que tenho grande apreço. Tenho um domínio rudimentar da das operações basicas e faço procedures e views. Ja trabalhei em projetos de fins educativos com o MySQL, SQL Server e o PostgresSQL

Java11

Escreva uma vez. Rode em qualquer lugar! Java é uma linguagem fortemente tipada, relativamente facil de aprender e cabe um destaque para as IDEs mais populares: o IntelliJ, e NetBeans. Tenho em Java um projeto de estudo que é o motivo pelo qual comecei a programar: "O Simples". E falaremos dele mais pra frente...

C#

C Sharp é o queridinho do momento aqui em casa... tenho lido muito a respeito e tenho gostado bastante do que tenho lido. A IDE é fantastica e a linguagem tem tudo o que precisa ter numa linguagem orientada a objeto. Sera que teremos projetinho por aqui em breve?
